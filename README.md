# Lumen Bible Study

A modern web-first Bible study prototype inspired by deep-study tools like Logos, but scoped for a cleaner first version: Scripture reading, search, notes, commentary, word study, cross references, a reading-plan progress card, and a lightweight study assistant.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Scripture excerpts use the public-domain World English Bible.
- The study-desk photo is from Unsplash by Tim Wildsmith.
- The app uses an original name and visual direction; it does not copy Logos branding.
